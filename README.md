# Connect Four

ローカル環境のブラウザで遊ぶ、2人対戦用のコネクトフォーです。

## セットアップ

```bash
pnpm install
```

## 起動

```bash
pnpm dev
```

表示されたローカルURLをブラウザで開いてください。

## テスト

```bash
pnpm test
```

## ビルド

```bash
pnpm build
```

## ルール

- 7列 x 6行の標準盤面です。
- 赤が先手、黄が後手です。
- 同じ画面で交互に列を選んでコマを落とします。
- 縦、横、斜めのいずれかで4つ連続したプレイヤーが勝ちです。
- 全マスが埋まって勝者がいなければ引き分けです。
